# 🛒 Django Ecommerce Project

A simple ecommerce web application built using Django.

---

## 🚀 Features

- 🛍️ Product listing
- 🛒 Cart system
- 🔐 Authentication (Login / Signup)
- ⚙️ Custom Admin Panel
- 📄 Pagination

---

## 🧰 Tech Stack

- Python
- Django
- HTML, CSS
- SQLite

---

## ⚙️ Setup Instructions

```bash
git clone https://github.com/harshitrajoriya/ecommerce.git
cd ecommerce

python3 -m venv venv
source venv/bin/activate

pip install -r requirements.txt

python manage.py runserver
